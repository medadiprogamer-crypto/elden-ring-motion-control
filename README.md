# Elden Ring Motion Control System

سیستم کنترل بازی Elden Ring با استفاده از حرکات بدن و دوربین لپتاپ

## 🎮 ویژگی‌ها

- ✅ تشخیص اسکلت بدن با استفاده از MediaPipe
- ✅ تبدیل حرکات بدن به دستورات دسته PS4
- ✅ پشتیبانی از تمام دکمه‌های اصلی Elden Ring
- ✅ کنترل دوربین با حرکات سر یا شانه‌ها
- ✅ حرکات حقیقی برای حمله و دفاع
- ✅ تشخیص خودکار حرکات

## 📋 نیازمندی‌ها

- Python 3.8+
- دوربین وب (لپتاپ)
- دسته PS4 (بی‌سیم)
- vigemebus (برای Windows)

## 🕹️ کنترل‌های حرکتی

### حرکت
| حرکت | فعالیت |
|------|--------|
| تکان دادن پاها در جای خود | راه رفتن |
| بالا آوردن زانوها | دویدن سریع |
| پریدن از زمین | پریدن |

### حمله و دفاع
| حرکت | فعالیت |
|------|--------|
| تکان دادن دست آرام | حمله سبک (X) |
| تکان دادن دست قوی | حمله سنگین (Y) |
| خم شدن به جلو | جاخالی دادن (B) |

### جادو و آیتم
| حرکت | فعالیت |
|------|--------|
| بالا آوردن هر دو دست | سایت magic (RB) |
| دست به سمت دهان | استفاده از آیتم (LB) |
| دستان روی سینه | Gesture (START) |

### کنترل دوربین
**روش 1: حرکت سر (پیش‌فرض)**
- سر را به چپ/راست کج کنید = دوربین چپ/راست
- سر را بالا/پایین کنید = دوربین بالا/پایین

**روش 2: حرکت شانه‌ها**
- شانه‌ها را بالا/پایین کنید = کنترل دوربین
- در `config.py` تغییر دهید: `CAMERA_CONTROL_MODE = 'shoulders'`

## ⚙️ تنظیم‌ها

فایل `config.py` را ویرایش کنید:

```python
# حساسیت تشخیص حرکات
WALK_THRESHOLD = 0.05
RUN_THRESHOLD = 0.15
JUMP_THRESHOLD = 0.3
ATTACK_LIGHT_THRESHOLD = 0.2
ATTACK_HEAVY_THRESHOLD = 0.4
DODGE_THRESHOLD = 0.25

# کنترل دوربین
CAMERA_CONTROL_MODE = 'head'  # یا 'shoulders'

# صاف‌کردن حرکات (0-1، بالاتر = نرم‌تر)
MOTION_SMOOTH_FACTOR = 0.7

# نمایش اطلاعات
DISPLAY_FPS = True
DISPLAY_SKELETON = True
DISPLAY_ACTIONS = True
```

## 🎓 نکات استفاده

1. **Calibration**: در ابتدا برنامه، لطفاً به صورت طبیعی ایستاده باشید
2. **نور خوب**: اتاقتان نور کافی داشته باشد
3. **فاصله از دوربین**: حدود 1-2 متر از دوربین فاصله بگیرید
4. **لباس متناسب**: لباسی بپوشید که بدن شما واضح باشد
5. **شمشیر مجازی**: یک لوله کوتاه را در دست بگیرید (اختیاری)
